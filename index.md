## What is the Ritter Ultimate Event Spawner?

The **Ritter Ultimate Event Spawner** is a powerful plugin for RPG Maker MV and MZ that allows developers to dynamically spawn, transform, and manage events in-game utilizing Object Pooling logic to recycle unspawned events.
With support for regions, terrain tags, and saved/temporary events, it gives full control over event placement, behavior, and lifecycle.

Key features include:

* Spawn events at specific coordinates or randomly based on regions or terrain tags.
* Transform existing events into new templates dynamically.
* Save events for persistent gameplay or use temporary events.
* Full control over event unspawning and cleanup.
* Store spawned Event IDs directly into game variables for further scripting logic.
* Event recycling for sustainable peak gameplay performance.
